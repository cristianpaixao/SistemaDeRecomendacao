# Sistema de Recomendação

[![Mahout](https://mahout.apache.org/images/mahout-logo-poweredby-55.png)](https://mahout.apache.org/)

Esse projeto é uma implementação básica de um sistema recomendações utilizando a biblioteca mahout. Com essa implementação você conseguirá:

  - Ler "data models" em csv 
  - Gerar recomendações baseadas no "data model"
  - Descobrir a margem de erro do algoritmo baseada no seu modelo de dados
